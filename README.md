# 🤖 AI Workplace Productivity Assistant

## 📌 Project Overview

The **AI Workplace Productivity Assistant** is a web-based application designed to improve workplace efficiency by automating common administrative tasks using Artificial Intelligence. The application helps professionals save time, improve communication, and stay organized through an intuitive dashboard interface.

This project was developed as part of the **CAPACITI AI Skill Accelerator Programme** and demonstrates the practical application of AI tools, prompt engineering, and responsible AI practices in solving real-world workplace challenges.

---

## 🎯 Problem Statement

Many professionals spend a significant amount of their workday performing repetitive tasks such as writing emails, planning daily schedules, and searching for information. These manual processes reduce productivity and leave less time for high-value work.

The AI Workplace Productivity Assistant addresses this problem by providing intelligent AI-powered tools that automate these tasks quickly and accurately.

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Workplace-Productivity-Assistant.git
```

### 2. Navigate to the Project Folder

```bash
cd AI-Workplace-Productivity-Assistant
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Create an Environment File

Create a `.env` file in the root directory and add your OpenAI API key:

```env
VITE_OPENAI_API_KEY=your_openai_api_key
```

---

### 5. Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 🚀 How to Use

### Smart Email Generator
1. Open the Email Generator page.
2. Enter the recipient, subject, purpose, and preferred tone.
3. Click **Generate Email**.
4. Review, edit, and copy the generated email.

### AI Task Planner
1. Navigate to the Task Planner.
2. Enter your tasks and working hours.
3. Click **Generate Schedule**.
4. Review or edit the generated plan.

### AI Chatbot
1. Open the Chatbot page.
2. Type a workplace-related question.
3. Receive an AI-generated response.
4. Continue the conversation as needed.
